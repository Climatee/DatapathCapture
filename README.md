# DatapathCapture
shell scripts intended for capturing a Datapath directshow video stream via CLI

Capture_NTSC:
-simply captures the datapath video stream via directshow and outputs it to a file: "output.mkv"
-uses libx264

Capture_PAL:
-simply captures the datapath video stream via directshow and outputs it to a file: "output.mkv"
-uses libx264
